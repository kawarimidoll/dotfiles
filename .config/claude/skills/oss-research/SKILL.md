# OSS Research

OSS リポジトリやライブラリの調査を行うスキル。
調査の種類に応じて、以下のサブエージェントを Agent ツールで **並列に** 起動する。

## サブエージェント

| エージェント | 用途 |
|---|---|
| `deepwiki-researcher` | リポジトリの概要・設計・使い方 |
| `source-researcher` | ソースコードを取得して実装を直接読む |
| `context7-researcher` | ライブラリの API ドキュメント・使用例 |

## 原則

- 必ず Agent ツールでサブエージェントとして実行すること（メインコンテキストの節約のため）
- 複数のサブエージェントが必要な場合は並列に起動すること
- サブエージェントの結果を簡潔に統合して報告すること
